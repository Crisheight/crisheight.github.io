  • Problems and Challenges

    The DarkSky API correlated to the skycons icon pack has been boughtout by Apple and no longer available, meaning I had to implement a different API and adjust the skycons pack code to match.
    A bigger problem is the openAPI unique key is exposed if I upload it to Github because of how the API is called.

  • Solutions

    OpenweatherAPI saved the day, and looking at their own icon pack code/correlations I was able to match the skycons pack to the proper weather conditions.
    In terms of protecting API keys, I would need to implement dotenv dependency. It might also just be easier to rewrite this as a react app since it has that built in. Since this is client side checking it seems the key is easier to dig up than if it was server side.
